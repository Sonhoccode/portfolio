# Portfolio (React + Vite)

Dự án portfolio cá nhân xây dựng bằng React và Vite, tập trung vào giao diện hiện đại, chuyển động mượt và hỗ trợ đa ngôn ngữ.

## Tính năng chính
- Giao diện portfolio một trang với điều hướng mượt.
- Hỗ trợ đa ngôn ngữ bằng i18next.
- Hiệu ứng chuyển động và cuộn (AOS, GSAP, Framer Motion).
- Thiết kế responsive.

## Công nghệ sử dụng
- React 19 + Vite 7.
- Tailwind CSS + PostCSS.
- React Router.
- i18next + react-i18next.
- MUI Icons.

## Cấu trúc thư mục
- `src/` mã nguồn chính.
- `src/components/` các component UI.
- `src/layouts/` bố cục trang.
- `public/locales/` file ngôn ngữ.
- `src/index.css` style và biến theme.

## Cài đặt
```bash
npm install
```

## Chạy môi trường phát triển
```bash
npm run dev
```

## Build production
```bash
npm run build
```

## Xem bản build
```bash
npm run preview
```

## Lint
